# Estudos IFSul (2015-2017)

Este repositório contém exercícios e códigos desenvolvidos durante meu período como estudante no IFSul de 2015 a 2017. 

O objetivo é preservar meu histórico de aprendizado e a evolução da minha lógica de programação.

## 📂 Tecnologias Encontradas
- **Arduino**: Prototipagem e controle de componentes eletrônicos.
- **Visualg**: Exercicios de aulas de lógica e algoritmos.
